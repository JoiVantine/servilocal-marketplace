ServiLocal

ServiLocal é uma plataforma que conecta clientes a prestadores de serviços de forma simples e rápida.

O objetivo é facilitar a contratação de profissionais locais em diversas categorias, permitindo que clientes encontrem prestadores qualificados e que profissionais ampliem sua presença digital.

Funcionalidades
Cadastro e autenticação de usuários
Perfis de cliente e prestador
Publicação de solicitações de serviço
Busca por categorias de serviços
Gestão de perfil profissional
Upload de imagens
Sistema de avaliações (em desenvolvimento)
Design responsivo para desktop e mobile
Tecnologias
React
TypeScript
Supabase
Tailwind CSS
Vite
Objetivo

Criar uma experiência simples e acessível para contratação de serviços locais, reduzindo a distância entre quem precisa de um serviço e quem pode executá-lo.

Status

Projeto em desenvolvimento contínuo.
